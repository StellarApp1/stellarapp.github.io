# StellarApp GitHub Pages

This repository hosts the public GitHub Pages site for **StellarApp**, the European platform for employment, professional reputation and verified work reviews.

**Official Website:**  
https://www.stellar-app.net

---

## About StellarApp

StellarApp connects **employees** and **employers** across Europe through a transparent reputation system based on:

- Verified professional reviews  
- Work history and skills  
- Employer recognition  
- Job applications through a unified platform  
- Merit-based visibility for candidates  

The platform supports **SDG 8 Decent Work and Economic Growth**, promoting fair employment and trustworthy hiring processes.

## Why GitHub Pages?

Hosting a public page on GitHub allows:

- Transparent documentation

- An open reference for developers, researchers and employers  
- Faster dissemination of the StellarApp mission
